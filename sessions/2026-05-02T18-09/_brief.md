# 📋 작업 브리프

**원 명령:** '모순 보고서'와 '가설 생성 UI'의 시각적 프로토타입을 제작

## 요약
기획된 게임의 핵심인 '모순 보고서'와 '가설 생성 UI'를 시각적으로 구현하기 위해 디자인과 개발 에이전트가 협업하여 프로토타입을 제작합니다.

## 분배
- **🎨 Designer**: TLA(시간, 장소, 금액) 데이터 구조와 '모순 보고서'의 차갑고 고풍스러운 스타일을 결합한 UI 레이아웃을 설계하세요. 특히 증거 간의 모순이 드러나는 시각적 연출과 가설 생성을 위한 인터랙티브한 UI 디자인 브리프를 작성하세요.
- **💻 Developer**: designer의 비주얼 브리프를 바탕으로, TLA 데이터 파싱 로직이 포함된 '모순 보고서' 화면과 유저가 가설을 입력/조합하는 '가설 생성 UI'의 프론트엔드 프로토토타입 코드를 작성하세요. 핵심 엔진(백엔드)과 시각적 요소를 결합해야 합니다.
- **✍️ Writer**: 프로토타입에 들어갈 텍스트(증거물 설명, 모순 발견 시의 경고 문구, 가설 입력용 플레이스홀더 등)를 게임의 분위기에 맞춰 몰입감 있게 작성하여 개발자에게 전달하세요.
